# Light-Control
这是一个根据Arduino IDE软件写的由光明电阻控制LED灯的实验，使其在光照暗的条件下，自动亮起。
使用材料有Arduino UNO的主板、连接主板USB的线、面包板、五根短的杜邦线、两个10k的电阻、一个LED的小灯。
